# Capistrano::Anycable

AnyCable integration for Capistrano.

## Installation

Add this line to your application's Gemfile:

```ruby
gem "capistrano-anycable", group: :development
```

And then execute:

```
$ bundle
```

## Usage

```ruby
# Capfile
require "capistrano/anycable"
```
